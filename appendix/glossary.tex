\chapter{Glossary}


\begin{description}
   \item[AgileZen] web-based software for project management based on agile methodologies.
   \item[Android] mobile operating system.
   \item[Android Debug Monitor]
   \item[Android Virtual Device] device configuration that is run with the Android emulator.
   \item[Backbone.js] JavaScript library for structuring javascript code
   \item[CMS] Construction and Management Simulation, a game in which players manage communities or projects with limited resources.
   \item[CSS] Cascading Style Sheets, a style sheet language used for describing the presentation semantics of an application. 
   \item[Dropbox] a web-based file hosting service.
   \item[Gantt diagram] a chart that illustrates a project schedule.
   \item[Git] a distributed version control system.
   \item[GitHub] a web-based hosting service for software development project, using Git for version control.
   \item[GUI] Graphical User Interface, allows users to interact with devices through graphical elements.
   \item[HTML] HyperText Markup Language, markup language for creating web pages and applications.
   \item[iOS] mobile operating system developed by Apple.
   \item[JavaScript] programming language
   \item[jQuery UI] JavaScript library for XXX
   \item[LaTex] a document preparation system and document markup language.
   \item[Native] XXX
   \item[Phonegap] mobile development framework, allowing programmers to build applications for mobile devices using JavaScript, HTML, and CSS.
   \item[Scrum] iterative and incremental methodology for managing software projects.
   \item[SDK] Software Development Kit, set of software development tools that allows for the development of applications.
   \item[Skype] voice-over-IP service and instant messaging client.
   \item[Sprint] the project is made up of time-boxed sprints with a delivery at the end.
   \item[Sprint Backlog] the list of work the team must address during the next sprint.
   \item[UML] Unified Modeling Language, a standardized, general-purpose modeling language in software engineering. 
   \item[Underscore.js] Javascript library with a lot of functional programming support.
   \item[Waterfall] sequential methodology for managing software projects.
